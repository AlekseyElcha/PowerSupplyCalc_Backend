**PSU Calculator - Калькулятор мощности блока питания ПК**

Инструкция по работе:
1) Перейти в Releases в этом репозитории.
2) Выбрать актуальную стабильную версию.
3) Скачать архив PSU_CalculatorV<номер>.zip - самый верхний в списке файлов.
4) Распаковать архив на ПК.
5) Перейти в папку и найти в ней .exe-файл (он единственный). Запустить его.

Внимание! В ряде случаев ОС Windows может пытаться блокировать запуск нашей программы!
**Наша программа не содержит вредоносного кода и не представляет опасности!**

Доступный функционал:
- Выбор ЦПУ, ГПУ, форм-фактора МП, ОЗУ, накопителей, привода, СВО
- Персональный выбор запаса по расчётам (в процентах)
- Получение рекомендаций по выбору БП по расчётам
- Возможность давать расчётам названия и сохранять их на ПК в .txt
